## Problem
Independent merchants needed an offline-capable POS with hardware integrations and reliable sync.

## Approach
Built a React Native app with offline-first data handling, Postgres backend, GraphQL (Apollo), and custom native C++ modules for hardware integrations via TurboModules.

## Impact
Streamlined operations and ensured stable performance for 20+ businesses under real-world usage.
